# imdb-sentiment-analysis
# IMDB Sentiment Analysis

This project focuses on sentiment analysis using the IMDB movie reviews dataset.
The project compares traditional machine learning models with the ALLAM Large Language Model (LLM) for classifying movie reviews as positive or negative.

## Models Used
- Logistic Regression
- Support Vector Machine (SVM)
- Naive Bayes
- ALLAM LLM

## Technologies
- Python
- Scikit-learn
- NLTK
- TF-IDF
- Transformers
- Matplotlib

## Project Workflow
1. Data preprocessing
2. Text cleaning and tokenization
3. Stopword removal
4. TF-IDF vectorization
5. Model training and evaluation
6. Performance comparison

## Results
- Logistic Regression achieved the highest accuracy: 85.4%
- Traditional ML models outperformed the ALLAM model in this experiment.

## Dataset
IMDB Movie Reviews Dataset from Hugging Face.
